HPBOX - Admin lançar/editar WOD por atleta v2

Alteração principal:
- A zona Resultados deixou de listar todos os atletas com formulário aberto.
- Agora existe apenas uma caixa "Lançar / editar WOD".
- O Admin escolhe o atleta num seletor, mete o resultado e guarda.
- A lista abaixo mostra apenas atletas que já têm força ou WOD registado nesse dia.

Como aplicar:
1. Substituir app.js e styles.css pelos ficheiros deste ZIP.
2. Fazer Commit changes no GitHub.
3. Esperar o Cloudflare Pages fazer deploy.
4. Abrir a app com ?v=admin-wod-2 para evitar cache.

Exemplo:
https://hpbox2.pages.dev/?v=admin-wod-2
